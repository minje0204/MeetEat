- **화상수업 [민상규]**
  - 중고등 수업, 대학교 강의를 위한 화상 강의 서비스
  - 출석체크 기능
    - 그룹 단위로 방을 개설하며 입장 시 출석이 반영된다
  - 퀴즈 출제 기능
    - 호스트가 퀴즈를 만들면 수강생들이 퀴즈를 푼다
    - 결과가 호스트에게 전달된다
  - 설문조사
    - 설문조사 결과가 호스트에게 전달된다
  - 전체 음소거
  - 수강생 응답 기능

  - 중,고등 / 대학교 분리
  - 대학 학사일정 학과일정
