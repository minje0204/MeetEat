- **공유 일기 화상미팅**
    - 하루를 마무리하면서 각자 있었던 일, 감정, 일상 등을 공유하는 서비스
    - 기능
        - 돌아가면서 지정한 시간만큼 발언 가능
        - 발언자 외 사람들은 위로, 축하, 공감, 반응할 수 있음 (웹엑스나 인스타 라이브 같이)
        - 랜덤 또는 지정된 그룹으로 참여 가능
        - 기본으로 익명으로 설정 (닉네임, 얼굴 필터, 목소리 필터, … 등)
        - 소수 인원(3~4명)으로 구성
        - 화상 미팅이 종료되면 하루의 기분을 이모지로 선택
        - 캘린더 형식으로 기분을 모아 볼 수 있음 + 텍스트 기록 가능
        - 1년 전, n년 전 오늘의 일기와 기분을 볼 수 있음 (구글 포토 같이)

**-코치님의 피드백-**

- 사생활에 관한 서비스는 **보안** 작업이 중요하다.
- 감성, 디자인에 중점을 두는 것이 적합할 것 같다.
- 서비스만의 컨텐츠를 만들어는 것이 좋다. (ex. 오늘의 질문 등)
- 기능 확장하려면 TODO 리스트 체크 또는 작성하는 것이 어떤지 제안해주셨다.
- 사용성을 고려하면 **모바일이 더 적합**할 것이라는 의견.
